Administrative Elements of the DOI Kernel

The table below lists the descriptive elements used in DOI Metadata Kernel.

 

Kernel element

Occurs

Description

registrationAuthorityCode

1

Code assigned to denote the name of the agency (authorized by the ISO 26324 Registration Authority) that issued this DOI name.

issueDate

1

Date when this DOI name was issued.

issueNumber

0-1

Number or other designation associated with the specific version of the DOI Kernel Metadata Declaration.
