DOI Proxy Service Characteristics

The DOI Proxy is accessible over IPv6, and supports DNSSEC.

The DOI Proxy consists of multiple servers running at multiple locations, with the load distributed evenly across all servers.

NOTE  The core DOI name resolution service is used by the DOI Proxy but is not constrained by the DOI Proxy.
